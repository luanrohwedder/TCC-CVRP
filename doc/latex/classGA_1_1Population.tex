\doxysubsection{Detailed Description}
Represents a \mbox{\hyperlink{classGA_1_1Population}{Population}} in genetic algorithm. 

A populations is a collection of Individuals, where each individual represents a solution. The populations has a defined m\+Size and keep track of the current m\+Generation.
